## Model fraction addition

How many quarters are pictured below: one quarter plus two more quarters?

$$\frac{1}{4} + \frac{2}{4} = \frac{3}{4}$$

Quarters are really fractions of a dollar — another way to say fourths.
One quarter plus two quarters equals three quarters. We can model the
same fact with fraction circles: start with one $\tfrac{1}{4}$ piece, add
two more $\tfrac{1}{4}$ pieces, and the result is $\tfrac{3}{4}$ of a
whole circle. So again, $\tfrac{1}{4} + \tfrac{2}{4} = \tfrac{3}{4}$.

**Example.** Use a model to find the sum $\tfrac{3}{8} + \tfrac{2}{8}$.

Start with three $\tfrac{1}{8}$ pieces. Add two more $\tfrac{1}{8}$
pieces. Count how many $\tfrac{1}{8}$ pieces there are in total: five.
The model shows that $\tfrac{3}{8} + \tfrac{2}{8} = \tfrac{5}{8}$.

## Add fractions with a common denominator

The example above shows that to add same-size pieces — meaning fractions
that already share a denominator — we simply add the number of pieces.

{{< callout type="info" >}}
  **Fraction addition.** If $a$, $b$, and $c$ are numbers where
  $c \neq 0$, then

  $$\frac{a}{c} + \frac{b}{c} = \frac{a+b}{c}$$

  To add fractions with a common denominator, add the numerators and
  place the sum over the common denominator.
{{< /callout >}}

**Example.** Find the sum $\tfrac{3}{5} + \tfrac{1}{5}$.

Add the numerators and place the sum over the common denominator:
$\tfrac{3+1}{5} = \tfrac{4}{5}$.

{{< fillin
  question="Find the sum: $\tfrac{3}{6} + \tfrac{2}{6}$"
  answer="\frac{5}{6}"
  answerDisplay="$\tfrac{5}{6}$"
  hint="Add the numerators and keep the same denominator."
>}}

**Example.** Find the sum $\tfrac{x}{3} + \tfrac{2}{3}$.

Add the numerators and place the sum over the common denominator:
$\tfrac{x+2}{3}$. Note this can't be simplified any further — since $x$
and $2$ aren't like terms, they can't be combined.

{{< fillin
  question="Find the sum: $\tfrac{x}{4} + \tfrac{3}{4}$"
  answer="\frac{x+3}{4}"
  answerDisplay="$\tfrac{x+3}{4}$"
  hint="Add the numerators over the common denominator; $x$ and $3$ are not like terms, so leave them separate."
>}}

**Example.** Find the sum $-\tfrac{9}{d} + \tfrac{3}{d}$.

Rewrite the first fraction with the negative sign in the numerator, since
$-\tfrac{a}{b} = \tfrac{-a}{b}$: $\tfrac{-9}{d} + \tfrac{3}{d}$. Add the
numerators: $\tfrac{-9+3}{d} = \tfrac{-6}{d}$. Rewrite with the negative
sign in front of the fraction:

$$-\frac{9}{d} + \frac{3}{d} = -\frac{6}{d}$$

{{< fillin
  question="Find the sum: $-\tfrac{7}{d} + \tfrac{8}{d}$"
  answer="\frac{1}{d}"
  answerDisplay="$\tfrac{1}{d}$"
  hint="Rewrite $-\tfrac{7}{d}$ as $\tfrac{-7}{d}$, add the numerators, then simplify the sign."
>}}

**Example.** Find the sum $\tfrac{2n}{11} + \tfrac{5n}{11}$.

Add the numerators and place the sum over the common denominator:
$\tfrac{2n+5n}{11}$. Combine like terms in the numerator:

$$\frac{2n}{11} + \frac{5n}{11} = \frac{7n}{11}$$

{{< fillin
  question="Find the sum: $\tfrac{3p}{8} + \tfrac{6p}{8}$"
  answer="\frac{9p}{8}"
  answerDisplay="$\tfrac{9p}{8}$"
  hint="Add the numerators (combining like terms), then place over the common denominator."
>}}

**Example.** Find the sum $-\tfrac{3}{12} + \left(-\tfrac{5}{12}\right)$.

Add the numerators and place over the common denominator:
$\tfrac{-3+(-5)}{12} = \tfrac{-8}{12}$. Simplify by removing the common
factor of $4$:

$$-\frac{3}{12} + \left(-\frac{5}{12}\right) = -\frac{2}{3}$$

  question="Find the sum: $-\tfrac{4}{15} + \left(-\tfrac{6}{15}\right)$"
  answer="-\frac{2}{3}"
  answerDisplay="$-\tfrac{2}{3}$"
  hint="Add the numerators over the common denominator, then simplify by removing common factors."
>}}

## Model fraction subtraction

Subtracting fractions with common denominators works much like adding
them. Think of a pizza cut into $12$ slices. Suppose five pieces are
eaten for dinner, leaving seven ($\tfrac{7}{12}$ of the pizza). If
Leonardo then eats two of the remaining pieces ($\tfrac{2}{12}$), five
pieces are left:

$$\frac{7}{12} - \frac{2}{12} = \frac{5}{12}$$

**Example.** Use fraction circles to find the difference
$\tfrac{4}{5} - \tfrac{1}{5}$.

Start with four $\tfrac{1}{5}$ pieces. Take away one $\tfrac{1}{5}$
piece. Count how many fifths are left: three. So
$\tfrac{4}{5} - \tfrac{1}{5} = \tfrac{3}{5}$.

  answer="\frac{3}{8}"
  answerDisplay="$\tfrac{3}{8}$"
  hint="Count how many eighth-pieces are left after taking some away."
>}}

## Subtract fractions with a common denominator

We subtract fractions with a common denominator in much the same way as
we add them.

{{< callout type="info" >}}
  **Fraction subtraction.** If $a$, $b$, and $c$ are numbers where
  $c \neq 0$, then

  $$\frac{a}{c} - \frac{b}{c} = \frac{a-b}{c}$$

  To subtract fractions with a common denominator, subtract the
  numerators and place the difference over the common denominator.
{{< /callout >}}

**Example.** Find the difference $\tfrac{23}{24} - \tfrac{14}{24}$.

Subtract the numerators and place the difference over the common
denominator: $\tfrac{23-14}{24} = \tfrac{9}{24}$. Simplify by removing
the common factor of $3$:

$$\frac{23}{24} - \frac{14}{24} = \frac{3}{8}$$

{{< fillin
  question="Find the difference: $\tfrac{19}{28} - \tfrac{7}{28}$"
  answer="\frac{3}{7}"
  answerDisplay="$\tfrac{3}{7}$"
  hint="Subtract the numerators, place over the common denominator, then simplify."
>}}

**Example.** Find the difference $\tfrac{y}{6} - \tfrac{1}{6}$.

Subtract the numerators and place the difference over the common
denominator: $\tfrac{y-1}{6}$. The fraction is already simplified, since
$y$ and $1$ are not like terms and cannot be combined.

{{< fillin
  question="Find the difference: $\tfrac{x}{7} - \tfrac{2}{7}$"
  answer="\frac{x-2}{7}"
  answerDisplay="$\tfrac{x-2}{7}$"
  hint="Subtract the numerators over the common denominator; $x$ and $2$ cannot be combined."
>}}

**Example.** Find the difference $-\tfrac{10}{x} - \tfrac{4}{x}$.

Rewrite the first fraction as $\tfrac{-10}{x}$. Subtract the numerators:
$\tfrac{-10-4}{x} = \tfrac{-14}{x}$. Rewrite with the negative sign in
front of the fraction:

$$-\frac{10}{x} - \frac{4}{x} = -\frac{14}{x}$$

{{< fillin
  question="Find the difference: $-\tfrac{9}{x} - \tfrac{7}{x}$"
  answer="-\frac{16}{x}"
  answerDisplay="$-\tfrac{16}{x}$"
  hint="Rewrite the first fraction as $\tfrac{-9}{x}$, subtract the numerators, then simplify the sign."
>}}

Now let's combine addition and subtraction in one expression.

**Example.** Simplify: $\tfrac{3}{8} + \left(-\tfrac{5}{8}\right) - \tfrac{1}{8}$.

Combine all the numerators over the common denominator:
$\tfrac{3+(-5)-1}{8}$. Simplify the numerator working left to right:
$3 + (-5) = -2$, then $-2 - 1 = -3$. Rewrite with the negative sign in
front:

$$\frac{3}{8} + \left(-\frac{5}{8}\right) - \frac{1}{8} = -\frac{3}{8}$$

{{< fillin
  question="Simplify: $\tfrac{2}{5} + \left(-\tfrac{4}{5}\right) - \tfrac{3}{5}$"
  answer="-1"
  hint="Combine all the numerators over the common denominator of $5$, working left to right."
>}}

## Key terms

**fraction addition** — for a common denominator $c \neq 0$,
$\tfrac{a}{c} + \tfrac{b}{c} = \tfrac{a+b}{c}$: add the numerators and
keep the denominator. **fraction subtraction** — for a common denominator
$c \neq 0$, $\tfrac{a}{c} - \tfrac{b}{c} = \tfrac{a-b}{c}$: subtract the
numerators and keep the denominator.
